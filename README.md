# Database Manager

## Описание

Готовый менеджер базы данных SQLite на Python, который можно переносить в любые проекты: от Telegram-ботов до веб-приложений.

Класс `DatabaseManager` берет на себя открытие и закрытие соединения с базой данных, выполнение SQL-запросов и получение результатов. Благодаря использованию контекстного менеджера (`with`) соединение всегда корректно закрывается, а изменения автоматически сохраняются.

## Возможности

* подключение к SQLite;
* автоматический `commit()` и `rollback()`;
* выполнение параметризованных SQL-запросов;
* выполнение запросов `SELECT`;
* использование через контекстный менеджер `with`;
* простой и понятный код, который легко расширять.

## Пример использования

```python
from database import DatabaseManager

with DatabaseManager("base.db") as db:
    db.execute(
        "INSERT INTO users(name, age) VALUES (?, ?)",
        ("Denis", 35)
    )

    users = db.select_all("SELECT * FROM users")
    print(users)
```
